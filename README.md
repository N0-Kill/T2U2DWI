# T2U2DWI
Proyecto Git / Flujo de Trabajo

Este es un proyecto web emulando el flujo de trabajo en Git/GitHub, simulando un entorno de desarrollo colaborativo.

Lider - Sebas
Documentador - Kilitium
Integrador - Nataly
Diseñador - N0-Ki11

## Flujo de Trabajo (Workflow)
Se utilizó el flujo **Git Feature Branch Workflow**:
1. La rama **main** contiene la versión estable.
2. Se crearon ramas de desarrollo (**dev** y **diseno**) para trabajar características por separado.
3. Se integraron los cambios mediante **Pull Requests** con revisión de código simulada.
